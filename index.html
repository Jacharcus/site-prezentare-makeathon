Chalk on concrete. For the 21st century

Playboard is a new way to draw on pavament, made for the 21st century. It can be used with a simple, disposable glove that can be produced for a very low price, so everyone can use it easily
It can be used to draw, color drawings and play games of varied difficulties.Thus, it can be used by kids ages 3-100 :)

Why Playboard?

  • Ease of acces

    By it's design, it can be used by any person who can move their fingers and who is able to see

  • Great for creativity

    Playing with Playboard is relaxing and stimulates everyone's creativity

  • Simply fun!

    No matter who you are, you will find Playboard a fun-to-use toy!

Playboard is used to color or to play stimulating games. One of these games is a memory game in which you have to remember the color of several LEDs, an another is a sound game in which, if you put pressure on a LED it creates a sound based on it's color.

Technology used

Playboard is builty in a sturdy, modular and simple way,
which makes it simple to install and repair if something breaks.

To create Playboard, we used simple and efficient technologies. It is built using LEDs, which are hard to break and install, pressure sensors and NFC readers.

Each of the gloves that are used to controll it has 5 NFC tags, one for each of your fingers. Thus, you can control the colors with your color coded glove :). All of this is powered by a tablet which controlls the whole system.
